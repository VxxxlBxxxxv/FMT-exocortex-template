# Ритуал ревизии избранного

> Выполняется после шага 6b (Требует внимания), перед записью DayPlan.
> Цель: не дать "избранному" превратиться в мусорный ящик.

## Шаг R1. Telegram Saved Messages

Вывести пользователю напоминание:

---
**📌 Telegram Saved Messages — ревизия**

Открой Saved Messages и пройдись по последним записям:
- Есть что-то, что нужно сделать сегодня? → добавь в план дня
- Есть что-то, что стоит зафиксировать как знание? → добавь в `fleeting-notes.md`
- Устаревшее / уже сделанное → удали

*Время: 2–3 мин*

---

Спросить пользователя: «Есть что добавить в план из Telegram?»

Если да → добавить в раздел «Требует внимания» или в план дня.
Если нет / нет ответа → продолжить.

## Шаг R2. YouTube Watch Later

Вывести пользователю напоминание:

---
**▶️ YouTube Watch Later — ревизия**

Загляни в плейлист «Смотреть позже»:
- Есть видео, которое актуально для текущих РП (перевод, IWE, AI)? → добавь в план как слот саморазвития
- Накопилось >20 видео? → добавить в «Требует внимания»: «Ревизия Watch Later (>20 видео)»
- Устаревшее / неинтересное → удали не задумываясь

*Время: 1–2 мин*

---

Спросить пользователя: «Есть видео для сегодняшнего саморазвития?»

Если да → добавить как слот в план дня (🟢, 0.5–1h).
Если нет / нет ответа → продолжить.
